**상황**: effect-typing 페이지가 사실상 네온 디자인을 보여주는 것이지 타이핑 이펙트 자체가 목적이 아님
**결정**: effect-typing 삭제 → design-neon 페이지로 대체. 사이버펑크 바 분위기, 쨍한 네온사인 핑크(#ff3a8c) + 광선검 블루(#5a9eff) + 퍼플(#c06aff). 히어로에 바닥 반사광 + 글자 깜빡임(flicker) 퇴폐미
**근거**: 디자인 트렌드 쇼케이스는 "왜"를 담아야 한다는 원칙. effect-blurry도 글래스모피즘과 중복되어 함께 삭제
**대안**: effect-typing 유지하고 별도 네온 페이지 생성 → 중복이라 기각

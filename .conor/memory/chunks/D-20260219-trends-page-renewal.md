**상황**: 트렌드 페이지가 비주얼 데모에 그치고 철학과 메시지가 부재
**결정**: Fluent UI 페이지처럼 각 디자인 컨셉을 살리되, 설명·철학·"왜 그래야만 하는지"를 함께 보여주는 구조로 리뉴얼
**근거**: 디자인 쇼케이스 프로젝트의 본질은 실험적이되 철학과 메시지를 담는 것. "디자인은 보는 것이 아니라 사용하는 것"
**대안**: 기존처럼 비주얼 데모 위주 유지 → 킥이 없고 카피처럼 보임

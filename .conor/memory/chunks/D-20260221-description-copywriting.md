**상황**: 각 트렌드 페이지의 description이 "Inspired by X — 특징 나열" 패턴으로 균일하고 밋밋했음
**결정**: 핵심 한 줄 + 위트 톤으로 전면 교체. 잘된 트렌드는 인정(찬사), 망한 트렌드는 왜 망했는지(비꼼)를 담음
**근거**: 사이드바에서 `text-overflow: ellipsis` 한 줄만 보이므로 앞부분이 핵심. 재치있는 한 줄이 긴 설명보다 효과적. 각 트렌드의 성패를 솔직하게 담아야 프로젝트 정체성("디자인 재해석")에 부합
**대안**: 전부 비판적 톤 → 잘한 건 인정해야 균형 잡힘. 한국어 → 사이트 전체가 영어이므로 영어 유지

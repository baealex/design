WINDOWS 95 ’95

It taught the world what a button looks like.

README.txt - Notepad

Windows 95's beveled borders made every clickable thing obviously clickable. Raised means push it. Sunken means type in it. A child could understand it. A billion people did. That was the strength: instant affordance.

The weakness? The 3D effects aged poorly. The gray surfaces became oppressive. The visual weight overwhelmed the content. But the principle underneath — that every element should communicate its function through its appearance — is eternal.

This page takes that principle and proves it still works. Not pixel-perfect retro emulation, but the affordance system itself — rebuilt for a modern context where raised surfaces, sunken fields, and window chrome aren't nostalgia. They're usable UI.

Principle 01

The Beveled Border

The defining visual trick of Windows 95. A white highlight on the top-left and a dark shadow on the bottom-right creates the illusion of a 3D surface lit from the upper left. Reverse the colors and the surface looks pressed inward. This single technique gave every pixel a physical meaning: raised means clickable, sunken means editable. No labels needed. Principle 02

The Window Chrome

Every application in Windows 95 lived inside a framed window. A blue gradient title bar told you what you were looking at. Three buttons in the corner gave you minimize, maximize, and close. A menu bar organized every action under predictable headings: File, Edit, View, Help. And a status bar at the bottom kept you informed. This chrome wasn't decoration — it was structure. Principle 03

The Start Menu

Microsoft's answer to "Where do I begin?" was revolutionary in its simplicity: a single button labeled "Start." Everything you could do with your computer funneled through one entry point. Programs, documents, settings, help — all organized in a cascading menu that rewarded exploration. The Start button became so iconic that it survived as a concept for thirty years. What This Is Not

  • ✗ Pixel-perfect retro emulation as visual gimmick
  • ✗ Gray boxes without interaction feedback
  • ✗ Skeuomorphism taken to absurdity
  • ✗ Nostalgia without function
  • ✓ 3D borders as instant affordance — "this is clickable"
  • ✓ Window chrome that frames and organizes content
  • ✓ Start menu pattern — one entry point to everything
  • ✓ Status bars that keep the user informed

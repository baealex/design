Aurora

The gradient is the design.
The UI stays out of its way.

Stripe proved that a mesh gradient can be a brand. Not a background detail — the main event. Bold, saturated color blobs drifting behind simple, solid UI elements.

Aurora UI is not Glassmorphism with a green tint. The panels aren't translucent — they're opaque and flat. It's not Neon with softer colors either. Nothing glows. Nothing emits light.

The gradient lives in its own layer. The UI lives in its own layer. They never compete. That separation is the entire point.

Principle 01

Mesh Gradients

Layer radial-gradient() blobs at different positions and sizes. Each blob is a single saturated color fading to transparent. Three to five layers is the sweet spot. Animate them on 15-25 second cycles — slow enough to feel organic, fast enough to notice if you stare.

  #0b0d1a;
animation: drift 18s ease-in-out infinite;
Principle 02

Layer Discipline

The gradient and the UI are separate layers. The gradient is atmospheric — it sets the mood. The UI is solid and opaque — it does the work. No backdrop-filter. No translucent panels. The cards don't borrow color from the gradient. They sit on top, cleanly, like signage on a painted wall.

Principle 03

Flat Interactions

Buttons are solid fills. Inputs are clean boxes. Hover states are subtle background shifts — not glow effects. The gradient already provides all the visual drama. Interactive elements just need to be readable and clickable. Let the mesh do the talking.

In Practice

Gradient as CTA

The mesh gradient isn't just a page background. Use it inside banners, hero sections, and call-to-action blocks. The gradient frames the message. The button is plain white. Maximum contrast, zero clutter.

What This Is Not

  • ✗ Glassmorphism — no backdrop-filter, no frosted panels
  • ✗ Neon — nothing glows, nothing emits light
  • ✗ Gradient on buttons or text — the gradient stays behind
  • ✗ Subtle tinted backgrounds — the mesh is bold and vivid
  • ✓ Mesh gradients as atmosphere, not decoration
  • ✓ Solid opaque UI — flat, readable, boring on purpose
  • ✓ Two layers: gradient layer and UI layer, never mixed
  • ✓ Slow animation — the gradient drifts, the UI stays still
